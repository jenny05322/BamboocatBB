# 竹貓 BB
這是一個對應竹貓人資系統的快速打卡小工具。只能在週一~週五使用，六日自己看著辦。

## 安裝
1. 下載檔案到電腦，並解壓縮
2. 開啟 chrome 瀏覽器
3. 開啟擴充功能
4. 將開發人員模式打勾
5. 點擊「載入未封裝擴充功能」按鈕，選擇第一步驟的資料夾

## 設定

### 選項
1. 點擊套件「選項」開啟設定頁面
2. 輸入竹貓人資系統的帳密
3. 點擊「儲存」按鈕

### 排程設定 (選擇性)
1. 開啟 windows 工作排程器
2. 建立新工作
3. 觸發程序選擇開啟電腦或登入時執行
3. 動作選擇「bat\bb_with_pc_start.bat」

## 使用
1. 點擊瀏覽器網址列旁的按鈕打卡
2. 如果有設定排程，則每天早上開電腦時會自動打卡
